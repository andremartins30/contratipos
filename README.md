# Calculadora de Contratipos

Este repositório contém o projeto da calculadora de contratipos.

## Estrutura do Projeto

- **/app**: Aplicação Next.js/React com a interface e lógica da calculadora.
- **/ _xlsx_extract**: Scripts e arquivos temporários de extração/análise de dados do Excel.
- **contratipo.xlsx**: Planilha original com as definições de contratipos e fórmulas.
